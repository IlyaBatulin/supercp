Description

Шаблон демонстрирует SaaS-компонент на Django для модели контроля доступа: пользователь вводит логин, пароль и ключ доступа, затем выбирает узел вычислительного кластера и вводит пароль узла. Результаты операций сохраняются в сессии браузера.

Если в бесплатной версии Replit недоступен отдельный Django-шаблон, создайте обычную Python-реплику, установите зависимости из requirements.txt и запустите проект командой python manage.py runserver 0.0.0.0:${PORT:-8000}.

Пользователь может изменить следующие переменные окружения:
